# ***Apa itu WIDGET ???***
```
Widget adalah aspek penting dari penyesuaian layar utama. Anda dapat melihatnya sebagai tampilan "sekilas" dari data dan fungsi
paling penting aplikasi yang dapat diakses langsung dari layar utama pengguna.
Pengguna dapat memindahkan widget ke seluruh panel layar utama, dan jika didukung, mengubah ukurannya untuk menyesuaikan jumlah
informasi di dalam widget sesuai keinginan. 
```
[Lihat Lebih...](https://developer.android.com/guide/topics/appwidgets/overview?hl=id)

# ***My Garden App & Widget***
##### ***Tampilan Awal***
```
Pada tampilan awal saat pertama kali aplikasi dijalankan, ketika user belum menambahkan tanamannya akan tampil halaman kosong
seperti dibawah ini. Untuk menambahkan tanaman, klik pada button + .

```
